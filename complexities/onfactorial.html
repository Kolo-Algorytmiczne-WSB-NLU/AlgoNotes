Szczegółowa złożoność

O(n!)

To koszt charakterystyczny dla pełnego generowania permutacji. Rośnie ekstremalnie szybko i działa tylko dla bardzo małych wartości n.

Wróć do przeglądu

Złożoności / O(n!)

Najkrótsza intuicja

Wszystkie permutacje

Jeśli Twoje rozwiązanie próbuje każdy możliwy układ elementów, niemal na pewno wpadasz w O(n!). To klasyczny koszt dla permutacji i części zadań backtrackingowych.

Typowe przykłady

  • generowanie wszystkich permutacji
  • backtracking po ustawieniach elementów
  • brute force dla bardzo małych danych

Typowe limity

Zwykle trzeba zostać przy n ≤ 12, a i tak nie zawsze będzie komfortowo.

Ważna uwaga

To jedna z tych klas, które komputer toleruje tylko wtedy, gdy dane wejściowe są naprawdę małe.

Przykład: permutacje przez swap
#include <stdio.h>

void swap(int* a, int* b) {
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

void permute(int arr[], int left, int right) {
    if (left == right) {
        for (int i = 0; i <= right; i++) {
            printf("%d ", arr[i]);
        }
        printf("\n");
        return;
    }

    for (int i = left; i <= right; i++) {
        swap(&arr[left], &arr[i]);
        permute(arr, left + 1, right);
        swap(&arr[left], &arr[i]);
    }
}

int main(void) {
    int arr[] = {1, 2, 3};
    permute(arr, 0, 2);
    return 0;
}
Przykład: backtracking po permutacjach
#include <algorithm>
#include <iostream>
#include <vector>

void permute(std::vector<int>& arr, int left) {
    if (left == static_cast<int>(arr.size())) {
        for (int x : arr) {
            std::cout << x << ' ';
        }
        std::cout << '\n';
        return;
    }

    for (int i = left; i < static_cast<int>(arr.size()); ++i) {
        std::swap(arr[left], arr[i]);
        permute(arr, left + 1);
        std::swap(arr[left], arr[i]);
    }
}

int main() {
    std::vector<int> arr = {1, 2, 3};
    permute(arr, 0);
    return 0;
}
Przykład: wszystkie ustawienia tablicy
using System;

class Program
{
    static void Permute(int[] arr, int left)
    {
        if (left == arr.Length)
        {
            Console.WriteLine(string.Join(" ", arr));
            return;
        }

        for (int i = left; i < arr.Length; i++)
        {
            (arr[left], arr[i]) = (arr[i], arr[left]);
            Permute(arr, left + 1);
            (arr[left], arr[i]) = (arr[i], arr[left]);
        }
    }

    static void Main()
    {
        int[] arr = { 1, 2, 3 };
        Permute(arr, 0);
    }
}
Przykład: generator permutacji
def permute(arr: list[int], left: int) -> None:
    if left == len(arr):
        print(arr)
        return

    for i in range(left, len(arr)):
        arr[left], arr[i] = arr[i], arr[left]
        permute(arr, left + 1)
        arr[left], arr[i] = arr[i], arr[left]


permute([1, 2, 3], 0)
