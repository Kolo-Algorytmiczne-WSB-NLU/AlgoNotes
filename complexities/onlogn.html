Szczegółowa złożoność

O(n log n)

To jeden z najważniejszych progów praktycznych. Wiele wydajnych algorytmów sortujących i rozwiązań divide & conquer kończy właśnie tutaj.

Wróć do przeglądu

Złożoności / O(n log n)

Najkrótsza intuicja

Sortowanie i dziel oraz zwyciężaj

W mainstreamowych językach domyślne sortowanie ma właśnie taki oczekiwany koszt. Jeśli do tego dochodzi merge liniowy po każdym poziomie podziału, także lądujesz w O(n log n).

Typowe przykłady

  • sort() i jego odpowiedniki
  • merge sort i heap sort
  • divide & conquer z liniowym scalaniem

Praktyczny obraz

Bardzo często jest to wystarczająco szybkie dla n ≤ 10^6, szczególnie w zadaniach z sortowaniem lub sweep line.

Klasyczny pattern

Wczytaj dane, posortuj i przejdź po nich jeszcze raz. To bardzo częsty szkic rozwiązania.

Przykład: qsort w C
#include <stdio.h>
#include <stdlib.h>

int cmp(const void* a, const void* b) {
    return (*(const int*)a - *(const int*)b);
}

int main(void) {
    int arr[] = {7, 1, 9, 3, 5};
    int n = 5;

    qsort(arr, n, sizeof(int), cmp); /* O(n log n) */

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}
Przykład: std::sort
#include <algorithm>
#include <iostream>
#include <vector>

int main() {
    std::vector<int> arr = {7, 1, 9, 3, 5};
    std::sort(arr.begin(), arr.end()); // O(n log n)

    for (int x : arr) {
        std::cout << x << ' ';
    }
    std::cout << '\n';
    return 0;
}
Przykład: Array.Sort
using System;

class Program
{
    static void Main()
    {
        int[] arr = { 7, 1, 9, 3, 5 };
        Array.Sort(arr); // O(n log n)

        Console.WriteLine(string.Join(" ", arr));
    }
}
