Szczegółowa złożoność

O(k log n)

Ten koszt pojawia się, gdy wykonujesz operację logarytmiczną wiele razy. Klasyczny przykład to top K elements albo kilka binary searchy na dużej strukturze.

Wróć do przeglądu

Złożoności / O(k log n)

Najkrótsza intuicja

K razy coś logarytmicznego

Najczęściej spotkasz to przy heap push/pop wykonywanym k razy albo przy binary search uruchamianym wielokrotnie dla różnych zapytań.

Typowe przykłady

  • top K elements przez heap
  • merge K sorted lists
  • binary search wykonywany K razy

Praktyczny obraz

Jeśli pojedyncze wyszukanie jest szybkie, ale musisz je zrobić wiele razy, koszt łączny zaczyna zależeć od obu parametrów: k i n.

Typowe limity

Taki koszt bywa komfortowy dla danych rzędu n ≤ 10^6, jeśli k nie jest zbyt duże.

Przykład: binary search wykonywany K razy
#include <stdbool.h>
#include <stdio.h>

bool contains(int arr[], int n, int target) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return true;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }

    return false;
}

int main(void) {
    int arr[] = {1, 4, 7, 10, 13, 18, 22};
    int queries[] = {7, 18, 2};
    int n = 7;
    int k = 3;

    for (int i = 0; i < k; i++) {
        printf("%d\n", contains(arr, n, queries[i]));
    }

    return 0;
}
Przykład: K wyszukiwań binarnych
#include <algorithm>
#include <iostream>
#include <vector>

int main() {
    std::vector<int> arr = {1, 4, 7, 10, 13, 18, 22};
    std::vector<int> queries = {7, 18, 2};

    for (int q : queries) {
        bool found = std::binary_search(arr.begin(), arr.end(), q);
        std::cout << found << '\n';
    }

    return 0;
}
Przykład: Array.BinarySearch wiele razy
using System;

class Program
{
    static void Main()
    {
        int[] values = { 1, 4, 7, 10, 13, 18, 22 };
        int[] queries = { 7, 18, 2 };

        foreach (int q in queries)
        {
            bool found = Array.BinarySearch(values, q) >= 0;
            Console.WriteLine(found);
        }
    }
}
Przykład: bisect dla wielu zapytań
from bisect import bisect_left

values = [1, 4, 7, 10, 13, 18, 22]
queries = [7, 18, 2]

for q in queries:
    pos = bisect_left(values, q)
    found = pos < len(values) and values[pos] == q
    print(found)
